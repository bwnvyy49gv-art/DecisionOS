# DecisionOS

DecisionOS is an open-source career and application planning system for students and early-career professionals.

## Vision

DecisionOS helps users make better decisions about jobs, internships, master programs, skills, and long-term career paths.

## Planned Features

- Application tracker
- Cover letter generator
- Job scoring and ranking
- Master program comparison
- Career self-test
- Skill gap analysis
- Personal career roadmap

## Tech Stack

First version:

- Python
- Streamlit
- SQLite
- Pandas

Later versions:

- FastAPI
- PostgreSQL
- React
- Tailwind CSS
- AI-based matching
